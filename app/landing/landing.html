Synrec Development Platform
  • To use this editor, place it inside the plugins folder of your project. The master editor will attempt to use NodeJS functions to modify custom JSON and/or text files related to Synrec plugins for RPG Maker.
  • The modified data will be read through a plugin to be installed in RPG Maker with the name Synrec_Master.js
  • Without that plugin, the settings made here will be entirely useless.
To begin, push the start button.
